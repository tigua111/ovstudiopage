# Origin Visuals — 品牌形象網站

靜態單頁範例，展示 Identity / Portfolio / Tech / Contact。使用 Tailwind Play CDN 快速開發（可選擇以 Tailwind CLI 建置進階版本）。

快速預覽

1. 在本機開啟 `index.html`（雙擊或以本機 HTTP server）。

建議進階工作流程

- 若要使用 Tailwind JIT 與生產建置：建立 `package.json` 並安裝 `tailwindcss`、`postcss`、`autoprefixer`，然後將 `src/css` 編譯到 `dist`。
